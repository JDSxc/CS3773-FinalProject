# CS3773-FinalProject
How to Set up XAMPP
1. Go to the XAMPP website or click on the link: https://www.apachefriends.org/.
2. Go through the setup until you reach 'Select Components'.
3. Select only Apache, MySQL, PHP, and phpMyAdmin.
4. Finish the setup.
5. Once finished, you will see a control panel to start the tools. Start Apache and MySQL.
6. To see if it was set up correctly, type localhost in your browser's URL, and you should see a dashboard page load up, which means it was set up successfully!

How to Clone a Repository for XAMPP (GitHub Setup)
1. Go to GitHub.
2. Click on the '<> Code' button.
3. Copy the HTTPS URL.
4. Go to where you installed XAMPP.
5. Go to the 'htdocs' folder and clone the repository here.
6. You should see a new folder named after the repository in 'htdocs'.
7. To test, just type 'localhost/CS3773-FinalProject' in your browser's URL (make sure Apache is running).
8. If everything was set up correctly, you should see an index page (or a homepage if we have an index file already).
